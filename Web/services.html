Services Provided in IAVMS Welcome to Digital Log System


Admin can add vehicle for his/her own Regiment.

Admin can update logbook for the vehicle of its own Regiment.

Admin can check major issues provided by driver.

Admin can see the part change made by the driver.

Admin can see the cardiary data which is updated by the driver.

Driver can update logbook,cardiary form the IAVMS app .

Driver can also see the near by units for any help.

Driver can also see his/her cardiary and logbook data.

Driver can notify his Regiment admin for any major issues.

Admin can change his own profile whenever needed.
